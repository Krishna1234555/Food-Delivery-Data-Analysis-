# Food-Delivery-Data-Analysis-
This is a comprehensive data analysis project designed for students learning data science, specifically focused on combining and analyzing data from multiple sources in different formats. The project simulates a real-world food delivery business scenario where data comes from different systems (CSV files, JSON APIs, and SQL databases).
