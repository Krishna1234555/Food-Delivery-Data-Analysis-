Food Delivery Data Analysis - Complete TutorialOverviewThis project demonstrates a complete data pipeline for analyzing food delivery data from three different sources:orders.csv - Transaction data (CSV format)users.json - User master data (JSON format)restaurants.sql - Restaurant master data (SQL format)The notebook combines these datasets and performs comprehensive exploratory data analysis (EDA).---Learning ObjectivesStudents will learn:Data Loading: How to load data from multiple formats (CSV, JSON, SQL)Data Merging: Performing relational joins using pandasData Cleaning: Handling actionable insights from data---Dataset Informationorders.csvContains: 10,000 ordersColumns:order_id: Unique order identifieruser_id: Foreign key to usersrestaurant_id: Foreign key to
